# Быстрый старт: от нуля до первой сессии за 15 минут

> **Для кого:** у тебя уже установлены Git, Node.js, GitHub CLI и Claude Code CLI.
> Если чего-то нет -- сначала пройди [Этап 0 из SETUP-GUIDE](SETUP-GUIDE.md).
> Не на macOS или не Claude Code? → **[PORTABILITY.md](PORTABILITY.md)**

---

## 1. Форкни и установи (5 мин)

> **Терминал** — это программа для ввода текстовых команд.
> - **macOS:** `Cmd + Пробел` → набери `Terminal` → Enter. Или: Finder → Программы → Утилиты → Терминал.
> - **Windows:** сначала установи [WSL](https://learn.microsoft.com/ru-ru/windows/wsl/install), затем Пуск → `Ubuntu` → Enter.

Папка `~/IWE` — это твоё рабочее пространство. Все репозитории IWE будут жить внутри неё. Именно эту папку ты откроешь в VS Code.

```bash
mkdir -p ~/IWE && cd ~/IWE
gh repo fork TserenTserenov/FMT-exocortex-template --clone
cd FMT-exocortex-template
bash setup.sh
```

Скрипт спросит:

- **GitHub username** — твоё имя на GitHub. Его видно в правом верхнем углу на github.com после входа в аккаунт. Например: `ivan_petrov`
- **Часовой пояс** — введи в формате `Europe/Moscow` (Москва), `Europe/Minsk` (Минск), `Asia/Novosibirsk` (Новосибирск), `Asia/Yekaterinburg` (Екатеринбург). Если не знаешь свою зону — спроси ИИ.

На остальные вопросы -- просто нажимайте Enter.

Если что-то непонятно - откройте другое окно браузера с claude.ai и спросите.

**Проверка:**
```bash
ls ~/IWE/CLAUDE.md && echo "OK: CLAUDE.md на месте"
ls ~/IWE/DS-strategy/ && echo "OK: DS-strategy создан"
```

## 2. Открой Claude Code (1 мин)

**Главный вариант — VS Code:**

1. Установи [VS Code](https://code.visualstudio.com), если ещё нет
2. В VS Code: нажми `Cmd+Shift+X` (macOS) или `Ctrl+Shift+X` (Windows) → найди расширение **Claude Code** от Anthropic → установи
3. `File → Open Folder` → выбери папку `~/IWE`
4. Нажми `Cmd+Shift+P` (macOS) или `Ctrl+Shift+P` (Windows) → введи **Claude Code: Open** → Enter

В правой части экрана появится панель чата с Claude — это и есть твоё рабочее окно.

**Альтернативный вариант — терминал:**
```bash
cd ~/IWE
claude
```

## 3. Первая сессия (9 мин)

**Убедись, что VS Code открыт с папкой ~/IWE:**

1. Запусти VS Code
2. `File → Open Folder` → выбери папку `~/IWE` (на macOS и Linux тильда `~` означает домашнюю папку; на Windows введи `C:\Users\твоё-имя\IWE`)
3. Если панель Claude Code не видна — нажми `Cmd+Shift+P` → **Claude Code: Open** → Enter
4. Внизу или справа появится панель чата — это поле ввода для общения с Claude

В поле ввода чата напиши:

> **Проведём первую стратегическую сессию**

(или явно `/strategy-session`, или любое из: «давай стратегировать», «strategy session», «открой стратсессию»)

Если Claude попросит разрешения на чтение файлов — нажми **Allow** (разрешить). Это нормально: ему нужно прочитать настройки и заметки в папке `~/IWE`.

Claude прочитает CLAUDE.md и memory/, затем проведёт через 4 шага:

1. **Цели** -- кем хочешь быть через год? Чему научиться?
2. **Неудовлетворённости** -- что мешает? Где разрыв?
3. **Первый WeekPlan** -- задачи на неделю с бюджетами
4. **Обновление MEMORY.md** -- рабочие продукты появятся в таблице

**Результат:** заполненные файлы в `DS-strategy/`:
- `docs/Strategy.md` -- твоя стратегия
- `docs/Dissatisfactions.md` -- неудовлетворённости
- `current/WeekPlan W{N}...md` -- план на неделю

> **Разработчикам T4+:** одна точка входа - `docs/developer/`. Открой её, и за 10 минут поймёшь конвейер разработки IWE и сделаешь первую задачу.

## 4. Начинай работать

Завтра утром Claude (в роли Стратега) сам подготовит план на день (DayPlan). Или попроси прямо сейчас:

> **Открывай день**

Claude соберёт информацию из всех репозиториев, календаря и заметок -- и предложит план.

Каждая рабочая сессия проходит по протоколу **ОРЗ**:

| Стадия | Что происходит | Ты делаешь |
|--------|---------------|------------|
| **Открытие** | Claude проверяет план и определяет задачу | Называешь задачу -- Claude согласует подход |
| **Работа** | Claude фиксирует ценные знания по ходу | Работаешь как обычно |
| **Закрытие** | Результат зафиксирован, план обновлён | Говоришь «закрывай» |
