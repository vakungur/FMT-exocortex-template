# Политика данных IWE

> **Версия:** 1.1 | **Дата:** 2026-03-27 | **Полная версия:** [DP.D.035](https://github.com/TserenTserenov/PACK-digital-platform/blob/main/pack/digital-platform/01-domain-contract/DP.D.035-data-policy.md)

Этот документ описывает, какие данные собирает IWE, где они хранятся и как ими управлять.

---

## 1. Какие данные собираются

Объём данных зависит от вашего тира:

| Тир | Что собирается |
|-----|----------------|
| **T0 Без аккаунта** | Ничего не собирается (нет входа через аккаунт) |
| **T1 Старт** | Telegram-профиль, имя, занятие, ответы марафона (~15 полей) |
| **T2 Изучение** | + лента, Q&A, история сессий, заметки, подписка (~40 полей) |
| **T3 Персонализация** | + цифровой двойник: цели, самооценка, рефлексия (~60 полей) |
| **T4 Созидание (IWE)** | + CLAUDE.md, memory/, extensions/, git-коммиты (~100+ полей) |

---

## 2. Где хранятся данные

IWE доставляется двумя способами: **дистрибутив** (Git + CLI, для продвинутых) и **платформа-хостинг** (одна кнопка, для новичков). Вырос — экспортируешь — владеешь.

| Что | Где | Кто контролирует |
|-----|-----|------------------|
| Профиль, сессии, тесты, ЦД | Neon PostgreSQL (cloud) | IWE (серверная часть) |
| Шаблон (методология, скрипты) | Git-репо `FMT-exocortex-template` | IWE (обновляется через `update.sh`) |
| `memory/`, `extensions/`, `params.yaml` (ваш контекст) | **Ваша машина + ваш приватный GitHub** | **Вы** |
| DS-strategy (ваша стратегия) | **Ваша машина + ваш приватный GitHub** | **Вы** |

> **IWE не имеет доступа к вашим личным файлам.** Ваши `memory/`, `extensions/`, DS-strategy существуют только на вашей машине и в вашем приватном GitHub-репо.

---

## 3. Что отправляется внешним сервисам

| Сервис | Какие данные | Частота |
|--------|-------------|---------|
| **Anthropic API** (Claude) | Промпт + профиль + текст вопроса | При каждом запросе |
| **GitHub** | Содержимое репозиториев | При push |
| **WakaTime** (опционально) | Метаданные (проект, язык, время). **НЕ** содержимое файлов | Постоянно |
| **MCP iwe-knowledge** (Gateway) | Поисковые запросы (текст). **Без** PII | При поиске |

Anthropic **не использует данные API для обучения** ([API Terms](https://www.anthropic.com/api-terms)).

---

## 4. Шаблон vs Ваши файлы

| | Шаблон (обновляемое) | Ваши файлы (личное) |
|--|---------------------|---------------------|
| Что входит | `setup.sh`, `update.sh`, `scripts/`, `seed/`, `roles/`, `CLAUDE.md §1-8` | `memory/`, `extensions/`, `params.yaml`, `CLAUDE.md §9-10`, DS-strategy |
| Кто пишет | IWE | Вы |
| Обновляется автоматически | Да (`update.sh`) | **Никогда** |
| Содержит ваши данные | Нет | Да |
| Удаляется при удалении аккаунта | Нет (это методология) | Нет (это ваши файлы) |

---

## 5. Ваши права

### Просмотр данных
Команда `/mydata` в боте показывает все собранные данные: профиль, активность, прогресс, консультации.

### Удаление данных
В боте введите: `Удалить. {Ваше имя}` — каскадное удаление из всех таблиц.

После удаления:
- Профиль, история, заметки, ЦД — удалены с серверной части
- `memory/`, `extensions/`, DS-strategy — остаются у вас (удалите сами при желании)
- Обезличенные агрегаты (DAU/MAU) — не удаляются

### Отзыв согласия
Вы можете в любой момент:
- Удалить данные в боте
- Удалить локальные файлы
- Отключить WakaTime, отозвать GitHub OAuth

---

## 6. Согласие

Запуская `setup.sh`, вы подтверждаете, что:

1. Ознакомились с тем, какие данные собираются
2. Понимаете, где хранятся данные и что отправляется третьим сторонам
3. Знаете, как просмотреть (`/mydata`) и удалить свои данные
4. Понимаете различие шаблона (обновляемое IWE) и ваших файлов (личное)

Продолжение использования IWE после обновления политики = согласие с новой версией.

---

## 6.1. Свобода данных

Все ваши данные хранятся в открытых форматах (Git, Markdown, YAML). Вы можете в любой момент:

- **Забрать всё** — `git clone` ваших репозиториев
- **Забрать ЦД** — экспорт цифрового двойника через бота
- **Обнулить данные** — полное удаление с серверной части
- **Сменить AI-вендор** — ваши файлы не привязаны к Anthropic
- **Развивать самостоятельно** — форк шаблона, свои скрипты, свои интеграции

---

## 7. Безопасность

### Рекомендации для вашей машины

- **FileVault** — включите шифрование диска
- **Firewall** — должен быть активен
- **Секреты** — только в `.env` (gitignored) или менеджере паролей
- **SIP** — не отключайте

### Self-hosted альтернативы

| SaaS | Self-hosted |
|------|------------|
| WakaTime | [Wakapi](https://github.com/muety/wakapi) |
| GitHub | [Gitea](https://gitea.io/) или [GitLab Self-Managed](https://about.gitlab.com/install/) |
