# Step 00: Открытие сессии

## ВДВ

| | |
|---|---|
| **Вход** | Черновик WeekPlan (`status: draft`), созданный session-prep. Флаг «первая сессия месяца?» |
| **Действие** | Проверить MonthClose gate. Показать сводку повестки. Спросить: «По повестке или срочное?» |
| **Выход** | Согласие пилота на повестку или запрос на срочное → переключение порядка шагов |

## Инструкция для Claude

1. **MonthClose gate (только для monthly-диспетчера):**
   - Если сегодня ≤7 числа месяца И в `sessions/` нет записи Strategy Session этого месяца:
     - Найди последний `archive/MonthClose YYYY-MM.md` (прошлый месяц)
     - Прочитай: мультипликатор, фаза, калибр, ревизия проектов, инсайты, R-вопросник, carry-over
     - Прочитай `archive/multiplier-trend.md` (если есть)
     - Сохрани сводку MonthClose для шага 04 (стратегическая сверка)
   - Если не первая сессия месяца → пропусти, сообщи: «MonthClose gate: skip»

2. **Показать сводку:**
   - Найди `WeekPlan W*.md` со `status: draft` в `current/`
   - Кратко: повестка, ключевые вопросы, carry-over
   - Если MonthClose gate сработал — добавь ключевые цифры (мультипликатор, фаза)

3. **Вопрос пилоту:**
   > «Начнём по повестке или есть срочные вопросы?»
   - Срочное → зафикси тему как нерегулярный блок (шаг 07). **СТОП — жди следующего сообщения.**
   - По повестке → скажи «Начинаем шаг 01». **СТОП — жди следующего сообщения пилота. Не читать файл шага 01 до следующего хода.**
