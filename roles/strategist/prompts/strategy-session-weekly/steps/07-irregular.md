# Step 07: Нерегулярные блоки

## ВДВ

| | |
|---|---|
| **Вход** | `docs/Session Agenda.md`, повестка сессии, срочные темы от пилота (шаг 00) |
| **Действие** | Обработать нерегулярные блоки: ретро, архитектура, разбор документа и др. |
| **Выход** | Результаты нерегулярных блоков зафиксированы |

## Инструкция для Claude

1. **Прочитать** `docs/Session Agenda.md`

2. **Определить применимые блоки:**
   - **Обход Backlog.md (обязательно):** прочитай `docs/Backlog.md` → активные записи. Hard-trigger сработал? → поднять как блок для обсуждения. Soft-trigger подошёл? → упомянуть как кандидата. Ни один не сработал → пропустить молча.
   - Ретро (если прошло ≥4 недель с последнего)
   - Архитектурное решение (если есть открытый ArchGate)
   - Разбор документа (если пилот принёс)
   - Срочные темы от пилота (если были на шаге 00)

3. **Если блоков нет:**
   - Сообщи: «Нерегулярных блоков нет — пропускаем». **СТОП — жди следующего сообщения. Не переходить к следующему шагу до ответа пилота.**

4. **Если блоки есть:**
   - Пройди каждый блок с пилотом
   - Зафиксируй результаты

5. **Вопрос пилоту:**
   > «Продолжить к утверждению / скорректировать / пропустить?»
